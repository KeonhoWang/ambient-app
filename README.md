# 🌿 Ambient Mix

A modern ambient soundscape mixer built with **React** that helps users create the perfect environment for focus, relaxation, studying, or sleep. By combining layered nature sounds with productivity tools like a **Pomodoro timer**, **weather-based recommendations**, and **listening statistics**, Ambient Mix provides a calming, privacy-first experience with no account required.

---

## ✨ Features

- 🌧️ Mix multiple ambient sounds
- 🎚️ Individual volume controls
- 💾 Save & load custom sound mixes
- 📤 Export & import mixes
- 🌤️ Weather-based sound recommendations
- ⏱️ Built-in Pomodoro timer
- 📊 Listening statistics
- 🎨 Multiple UI themes
- ✨ Animated particle background
- 🔒 Privacy-first local storage
- 📱 Responsive design

## ⚙️ How It Works

Ambient Mix allows users to combine multiple ambient sounds and control the volume of each independently to create a personalized listening experience. User preferences, saved mixes, themes, and listening statistics are stored locally using **LocalStorage**, ensuring complete privacy. The app also integrates the **OpenWeatherMap API** to recommend sound combinations based on the user's current weather while the built-in **Pomodoro timer** helps users stay productive.

---

## 🚧 Challenges & Solutions

### 🎵 Managing Multiple Audio Sources
Implemented independent audio instances with **Howler.js** so multiple sounds can play simultaneously with separate volume controls.

### 💾 Persistent User Data
Used **LocalStorage** to automatically save mixes, themes, and listening statistics between sessions.

### 🌤️ Dynamic Sound Recommendations
Integrated the **OpenWeatherMap API** to suggest relaxing sound combinations based on the user's local weather.

### 🎨 Creating an Engaging UI
Designed a responsive interface featuring multiple themes, animated particle backgrounds, and intuitive controls for an enjoyable user experience.
